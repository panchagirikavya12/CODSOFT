# Landing Page - CODSOFT Task 2
## About the Project
This responsive landing page was built as part of my CODSOFT Web Development Internship.
The design focuses on a modern User Interface with clear navigation and call-to-action sections.

## Live Demo
[View Live Page](https://panchagirikavya12.github.io/CODSOFT/Task-2-Landing-Page/)

## Technologies Used
- HyperText Markup Language 5 (HTML5)
- Cascading Style Sheets 3 (CSS3)

## Key Features
- Fully responsive design for mobile and desktop
- Navigation bar with smooth layout
- Hero section with heading and Call To Action button
- Clean typography and spacing
- Professional color scheme

## What I Learned
- Creating responsive layouts using Cascading Style Sheets
- Structuring a landing page with proper sections
- Using CSS for styling buttons, navbar and content
- Deploying a project using GitHub Pages

1. Clone this repository - Download this code to your computer
