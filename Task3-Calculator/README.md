# Basic Calculator - CODSOFT Task 3
## About the Project
This is a basic calculator built as part of my CODSOFT Web Development Internship.
It performs fundamental arithmetic operations with a clean and responsive design.

## Live Demo
[View Live Page](https://panchagirikavya12.github.io/CODSOFT/Task3-Calculator/)

## Technologies Used
- HyperText Markup Language 5 (HTML5)
- Cascading Style Sheets 3 (CSS3)
- JavaScript

## Key Features
- Addition, Subtraction, Multiplication, Division
- Clear and Delete functions
- Responsive design for mobile and desktop
- User-friendly button layout
- Real-time calculation display

## What I Learned
- Using JavaScript for calculations and logic
- Handling button click events
- Styling a calculator using CSS
- Connecting HTML, CSS and JavaScript files
- Deploying a project using GitHub Pages

## How to Run Locally
1. Clone this repository - Download this code to your computer
2. Open `index.html` in your browser
